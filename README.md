# Anuja — Product Designer Portfolio ✦

> *"Turning curiosity about people into thoughtful design."*

A personal portfolio for **Anuja**, a Product & UX Designer with 1.8 years of experience crafting human-centred digital experiences. Built entirely with vanilla HTML, CSS, and JavaScript — no frameworks, no dependencies — just intentional design and clean code.

## 🎨 Design Highlights

| Feature | Details |
|---|---|
| **Aesthetic** | Editorial grid-paper · Red serif · Cream background |
| **Typography** | Playfair Display (serif headlines) + Inter (body) |
| **Cursor** | Custom dual googly-eye cursor with iris tracking |
| **Background** | Animated floating UI wireframe shapes + morphing gradient blobs |
| **Interactions** | Magnetic buttons · 3D card tilt · Cursor spotlight · Click sparks |
| **Animations** | Scroll reveal · Hero image slideshows · Parallax inline photos |

---

## 🗂️ Sections

- **Hero** — Full-viewport headline with inline cycling photo slideshows
- **Selected Work** — 2×2 project card grid with hover lift effects
- **About** — Intro card with Resume & LinkedIn CTAs
- **Experience** — Clean timeline listing 7 internships (1.8 yrs)
- **What I Do** — Services: Product Design · UX Research · Website Design · Visual Design
- **Contact** — Call-to-action with email link
- **Footer** — Links, social, and email

## 🛠️ Tech Stack

```
HTML5          — Semantic markup, SEO meta tags
CSS3           — Custom properties, Grid, Flexbox, keyframe animations
Vanilla JS     — Canvas API, IntersectionObserver, custom cursor, parallax
Google Fonts   — Playfair Display + Inter
```

No build tools. No frameworks. Just a single `index.html` + `style.css` + `script.js`.
